API¶

class experiments.Experiment(name, kind, measure_func=None, ident='', verbose=False, base_path=None, **kwargs)¶
Parameters

  • kwargs – A list of keyword arguments to pass into the experiment. These can be scalars/strings, or lists. If lists, all cartesian products of all possible experiments will be run.

  • name – The full name of the specific experiment (i.e. LIA1-1)

  • kind – A string of the kind of experiment we are carrying out (i.e. XRD, simulation, etc.

  • measure_func – The function to execute which returns data given a particular experimental condition. Must take the experimental condition as an argument

  • base_path – The absolute or relative base path of all structures.

  • verbose – Verbose output enable/disable

generate_conditions(comb_type='cartesian', **factors)¶

Generates a list of desired conditions from the specified factors and their levels.

Parameters

  • factors – dictionary with the factors and their desired levels

  • comb_type – “cartesian” (outer product) - generates a full factorial experiment from the set of factors, “individual” creates an individual condition from the specified factor levels, “1to1” generates a 1-to-1 mapping of factor levels.

generate_groups(data_dict=None, group_along='replicate', grouping_type='name')¶

Generates unique groups for grouping data data by condition which have different replicates but the some condition otherwise

Parameters

  • data_dict – Dictionary of named data for which to generate groups

  • group_along – Condition name that the data should be grouped by

  • grouping_type – “name” or “value”. Allows generation of groups into groups which have the same condition and run over one final condition, or groups which vary the value of a single condition.

Returns groups
